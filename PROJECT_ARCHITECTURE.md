# ERP System - Project Architecture

---

# 1. Project Overview

## Project Name

Enterprise Resource Planning (ERP) System

## Description

A full-stack ERP application built using the MERN stack.

The application provides authentication, email verification, and role-based access. After authentication, users access a centralized dashboard that acts as the entry point for all ERP modules.
